# 開発言語
- Ruby 3.0.1
- rails 6.1.6

# 就職Termの技術
- お気に入り機能
- `gem devise`

# カリキュラム外の技術
- `gem gmaps4rails`
- `gem geocoder`
- `gem ransack`

# 実行手順
$ git clone git@github.com:2303-DIC-MEG/Coffee_farm.git  
$ cd Coffee_farm   
$ bundle install   
$ rails db:create && rails db:migrate   
$ rail s
